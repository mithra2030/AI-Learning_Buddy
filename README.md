# 🎓 AI Learning Buddy

## 📌 Project Overview

AI Learning Buddy is an interactive AI-powered learning assistant developed using **Python**, **Streamlit**, and **Groq AI**. The application helps users learn any topic in a simple and engaging way by providing concept explanations, real-life examples, quizzes, and answers to user questions.

---

## 🚀 Features

- 📖 Explain any topic in simple language
- 🌍 Generate real-life examples
- 📝 Create 5-question quizzes with answers
- 💬 Ask AI any question related to the topic
- 🎨 User-friendly Streamlit interface
- ⚡ Fast AI responses using Groq

---

## 🛠️ Technologies Used

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-Learning_Buddy.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure API Key

Create Streamlit Secrets and add your Groq API Key:

```
GROQ_API_KEY="YOUR_GROQ_API_KEY"
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
